### 简化规则

1. **相同 Arcana 合成** → 結果為該 Arcana (如 愚者+愚者 = 愚者)

2. **相鄰 Arcana 合成** → 結果為兩者之一 (如 愚者+魔术师 = 死神 或 命运)

3. **特殊組合**:
   - 愚者 + 战车 = 月
   - 魔术师 + 正义 = 太阳  
   - 女教皇 + 恋爱 = 女教皇
   - 女皇 + 战车 = 战车
   - 皇帝 + 节制 = 信念
   - 教皇 + 隐士 = 死神
   - 恋爱 + 太阳 = 正义
   - 战车 + 隐士 = 死神
   - 正义 + 塔 = 审判
   - 隐士 + 塔 = 顾问官
   - 命运 + 塔 = 命运
   - 力量 + 星星 = 审判
   - 倒悬者 + 太阳 = 顾问官
   - 死神 + 顾问官 = — (无)
   - 塔 + 顾问官 = — (无)

### 使用方式

合成價格 = 基礎價格 + 寶魔等級加成

**基礎價格公式**:
```
價格 = ( Persona A 等級 + Persona B 等級 ) × 10
```

**示例**:
- 亚森(Lv.1) + 迪亚哥(Lv.3) = 基础价格 (1+3)×10 = 40
- 加入摄政王(愚者+1) = 40 + 1 = 41

## 3. 合成价格计算公式

```
合成价格 = (素材A等级 + 素材B等级) × 10 + 宝魔加成
```

### 特殊价格规则

1. **等级差修正**:
   - 素材等级差超过 10 级时，较低者按 10 级计算

2. **宝魔加成**:
   - 每个宝魔提供表格中的等级修正值
   - 可以叠加多个宝魔

## 4. 技能继承规则

### 基础继承

- **二体合成**: 继承 1 个技能 (随机选择)
- **多体合成**: 继承 2-5 个技能 (根据合成数量)

### 技能继承概率

| 继承类型 | 条件 | 概率 |
|---------|------|------|
| 普通技能 | 自动继承 | 100% (1个) |
| 高阶技能 | 5% 概率额外继承 | 5% |
| 遗传技能 | 指定遗传 | 100% |

### 技能继承表

每个 Persona 有以下继承标志:
- `inherit_phys`: 物理
- `inherit_gun`: 枪
- `inherit_fire`: 火
- `inherit_ice`: 冰
- `inherit_elec`: 电
- `inherit_wind`: 风
- `inherit_psy`: 念
- `inherit_nuke`: 核
- `inherit_bless`: 祝福
- `inherit_curse`: 咒杀
- `inherit_abnormal`: 异常状态
- `inherit_recovery`: 恢复技能

## 7. AI 实现提示

### 计算合成结果的伪代码

```javascript
function calculateFusionResult(personaA, personaB, treasureDemons = []) {
  // 1. 获取 Arcana 索引
  const arcanaA = getArcanaIndex(personaA.arcana);
  const arcanaB = getArcanaIndex(personaB.arcana);
  
  // 2. 查表获取结果 Arcana
  const resultArcana = FUSION_TABLE[arcanaA][arcanaB];
  
  // 3. 计算等级
  let baseLevel = Math.floor((personaA.level + personaB.level) / 2);
  
  // 4. 应用宝魔加成
  for (const demon of treasureDemons) {
    baseLevel += TREASURE_MODIFIERS[demon.name][resultArcana];
  }
  
  // 5. 确保等级不低于 1
  baseLevel = Math.max(1, baseLevel);
  
  return {
    arcana: resultArcana,
    level: baseLevel,
    // 在该 Arcana 的等级范围内查找最接近的 Persona
    resultPersona: findClosestPersona(resultArcana, baseLevel)
  };
}
```

### 数据存储建议

```sql
-- 合成表 (22x22 矩阵)
CREATE TABLE fusion_table (
  arcana_a INT,      -- 0-21
  arcana_b INT,      -- 0-21
  result_arcanum INT -- 0-21, -1 表示无结果
);

-- 宝魔加成表
CREATE TABLE treasure_modifiers (
  treasure_name VARCHAR(20),
  arcana VARCHAR(10),
  level_modifier INT
);
```
